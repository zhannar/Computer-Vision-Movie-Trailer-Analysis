Specify image path in main.bat. Both relative path and absolute path are accepted(relative path preferred).
Run main.bat to obtain the biconcept responce of the image.
Result will be saved in ./result/imagename-biconcept.mat. It will be a 1200 length vector.
You need Matlab MCR for Matlab 2013a installed in your computer to run the binary. 
Download: http://www.mathworks.com/products/compiler/mcr/index.html
Be sure to set the path of the MCR either in system environment variables or before running the binary.
